

# THE NORO-CODER (Portfolio Website)

This is a personal portfolio website built with Next.js, React, and Tailwind CSS. The website showcases my work, skills, and background information. It is designed to be responsive and accessible across different devices and screen sizes.

## Technologies Used

- [Next.js](https://nextjs.org/): A React framework for building server-rendered React applications.
- [React](https://reactjs.org/): A JavaScript library for building user interfaces.
- [Tailwind CSS](https://tailwindcss.com/): A utility-first CSS framework for rapid UI development.
- [NextUI](https://nextui.org/): A modern React UI library with a set of high-quality components.
- [Framer Motion](https://www.framer.com/motion/): A production-ready motion library for React on the web.
- [Radix Icons](https://icons.radix-ui.com/): A crisp set of 15x15 icons for use in web and mobile apps.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/ankitmrmishra/TheNoroCoder.git
```

2. Install dependencies:

```bash
cd TheNoroCoder
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Building for Production

To build the production-ready version of the website, run:

```bash
npm run build
```

Then, to start the production server:

```bash
npm start
```
